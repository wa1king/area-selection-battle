# GitHub上传命令指南

## 步骤1：在GitHub上创建新仓库
1. 访问 https://github.com/new
2. 仓库名称：area-selection-battle（或您喜欢的名称）
3. 描述：选面积大作战 - 一个挑战视觉判断能力的HTML5游戏
4. 选择 Public
5. 不要勾选 "Add a README file"
6. 点击 "Create repository"

## 步骤2：执行以下命令（请替换YOUR_USERNAME和YOUR_REPO_NAME）

# 添加远程仓库
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 推送到GitHub
git branch -M main
git push -u origin main
