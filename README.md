# 选面积大作战 (Area Selection Battle)

一个基于HTML5和JavaScript的网页游戏，挑战玩家的视觉判断能力。

## 🎮 游戏介绍

在这个游戏中，矩形区域被分割成4个不同颜色的区域，它们的面积极其接近（差异≤2个格子）。玩家需要凭借视觉判断选择面积最大的区域。

## 🎯 游戏特色

- **30个关卡**：每个关卡都有独特的区域分割模式
- **精确算法**：使用"初始化+扰动+面积平衡"三步方案生成区域
- **视觉挑战**：面积差异极小，考验玩家的观察力
- **动画效果**：精美的揭晓动画和排名显示
- **响应式设计**：适配不同屏幕尺寸

## 🚀 技术特点

### 核心算法
1. **四象限初始化**：提供稳定的起始分割
2. **受控随机扰动**：打破矩形边界，增加不规则性
3. **面积平衡**：精确控制区域面积差异
4. **连通性检查**：使用BFS算法确保每个区域连通

### 技术栈
- HTML5 + CSS3
- 原生JavaScript (ES6+)
- SVG图形渲染
- 种子随机数生成器

## 🎲 游戏规则

1. 观察4个不同颜色的区域
2. 点击选择你认为面积最大的区域
3. 点击"揭晓"按钮查看结果
4. 区域会逐格消失，显示从小到大的排名
5. 完成30关即可通关

## 🛠️ 本地运行

1. 克隆或下载项目文件
2. 直接在浏览器中打开 `index.html`
3. 开始游戏！
